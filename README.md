# Holodos

**HOLODOS_CORE** — это **школьный проект**, разработка которого ведётся в рамках личного обучения и экспериментов с архитектурой REST API, контроллерами экрана и обработкой состояния устройства.  

Проект создаётся с целью изучить и закрепить практические навыки в построении модульного приложения с разделением логики, интерфейса и API.На текущем этапе проект находится в активной разработке, реализована базовая архитектура и пользовательский интерфейс.

## Текущее состояние проекта

На данный момент готово:

* **Ядро проекта**

  * Реализовано REST API
  * Ядро отвечает за логику, обмен данными и управление состоянием устройства

* **Экран (UI)**

  * Реализован экран с иконками
  * Интерфейс рассчитан на компактный дисплей(в моем случае 320x240)

## Архитектура

Проект построен по модульному принципу с чётким разделением ответственности между ядром (логикой), контроллерами и ресурсами. Основная идея — изолировать бизнес-логику от конкретной реализации экрана и датчиков.

### Общая схема

* **REST API** — центральная точка взаимодействия
* **Controllers** — слой управления экраном и датчиками
* **State** — единое хранилище состояния устройства
* **Resources (img)** — статические ресурсы интерфейса

* Создать удобный и быстрый интерфейс для управления устройством
